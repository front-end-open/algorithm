# 2.删除链表的倒数第 N 个节点

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。 进阶：你能尝试使用一趟扫描实现吗？

示例 1:

输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
1
2

示例 2：

输入：head = [1], n = 1
输出：[]
1
2

示列 3:

输入：head = [1,2], n = 1
输出：[1]
1
2

TIP

提示：

链表中结点的数目为 sz

  • 1 <= sz <= 30
  • 0 <= Node.val <= 100
  • 1 <= n <= sz

解答:

var removeNthFromEnd = function (head, n) {
  let current = head;
  let count = getLength(head);
  let pre = count - n;
  if (pre === 0) {
    return head.next;
  }
  for (let i = 0; i < pre - 1; i++) {
    current = current.next;
  }

  current.next = current.next.next;

  return head;
};

function getLength(head) {
  let current = head;
  let count = 0;
  while (current != null) {
    current = current.next;
    count++;
  }

  return count;
}

  1. 环形链表

给定一个链表，判断链表中是否有环。 如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。 为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。注意：pos 不作为参数进行传递，仅仅是为了标识链表的实际情况。 如果链表中存在环，则返回 true 。 否则，返回 false

进阶：你能用 O(1)（即，常量）内存解决此问题吗？

示范 1:

输入：head = [3,2,0,-4], pos = 1
输出：true
解释：链表中有一个环，其尾部连接到第二个节点。
